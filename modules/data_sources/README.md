# 数据源模块

## 已完成
- 定义统一接口 `DataSource.fetch(symbol, start, end)`，返回标准化的 OHLCV `DataFrame`。
- 提供离线可用的 `SyntheticSource`，生成随机游走数据并可设置随机种子，便于无网络时完成端到端流程。
- CSV 缓存工具：按照标的与日期范围生成路径，支持缓存写入/读取与简单的元数据记录。
- 数据校验：检查缺失、排序、重复索引与字段完整性，返回校验报告。

## 必须完成但未完成
- 接入 baostock 与 akshare 的真实数据拉取适配器，并与 `SyntheticSource` 共用校验/缓存流程。
- 多源交叉验证：在相同时间窗口内对比价量差异并产出差异报告。
- 数据版本化：为同一标的的不同数据源/复权方式添加版本标签与一致性校验。

## 可选项
- 接入其他数据源（tushare/聚宽等），共用统一接口。
- 增加数据质量评分与告警阈值配置。
- 将缓存从 CSV 升级为可选的本地数据库存储。
