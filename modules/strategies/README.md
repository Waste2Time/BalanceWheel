# 策略库模块（strategies）

## 已完成
- 抽象基类 `Strategy` 与示例策略：均线交叉、轮动、组合权重占位。
- 参数化配置（窗口、仓位等）以支持 baseline/variant 对比。
- 新增 `__main__` 入口，支持 `python -m modules.strategies` 查看用法提示。
