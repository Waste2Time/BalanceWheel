# 回测引擎模块

## 已完成
- 统一接口 `BacktestEngine.run(strategy, data, config)`，输出包含收益曲线、交易记录与绩效指标的结果对象。
- 简易本地引擎：支持按信号下单、持仓更新、手续费/滑点配置，以及基于最大回撤的止损退出。
- 基础绩效指标：累计收益、年化收益、夏普比率、最大回撤、胜率、平均盈亏。
- 与实验运行器的衔接：支持批量运行并返回标准化结果。
