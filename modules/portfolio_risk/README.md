# 组合与风控模块（portfolio_risk）

## 目标

- 统一组合管理与风险控制策略。
- 支持多资产与组合投资策略的约束管理。

## 模块职责

- 权重生成：等权/风险平价/自定义权重。
- 风险预算：最大仓位、行业暴露等约束。
- 再平衡：按频率或触发条件调整仓位。

## 风控指标

- 最大回撤（MDD）
- 波动率与目标波动率
- VaR / ES（可选）

## 与回测引擎接口

- 提供 `position_sizer` 与 `risk_check` 钩子。
- 回测中每次调仓前执行风险检查。

## 示例策略

- 等权组合
- 风险平价（Risk Parity）
- 最大回撤限制组合
